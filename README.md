# Data-Analysis-4th-year

This repository serves as a stash of all materials used for the discipline "Data Analysis for Sociology" taught for 4-th year students in Higher School of Economics

## The structure

### [Week1](https://github.com/Esquire-of-Rohan/Data-Analysis-4th-year/tree/main/Week1)

This week is devoted to data imputation methods


### [Week2](https://github.com/Esquire-of-Rohan/Data-Analysis-4th-year/tree/main/Week2)

This week is devoted to Decision Trees and Random Forest models

### [Week3](https://github.com/Esquire-of-Rohan/Data-Analysis-4th-year/tree/main/Week3)

This week is devoted to Poission regression

### [Week4](https://github.com/Esquire-of-Rohan/Data-Analysis-4th-year/tree/main/Week4)

This week is devoted to Quantile regression
